# Snippet Manager 利用ガイド

対象読者: アプリの利用者（開発者向けは [SETUP.md](SETUP.md)）

---

## 1. アプリの概要

Snippet Manager は **メニューバーに常駐** し、登録したテキスト（スニペット）と **クリップボード履歴** をホットキーから素早く呼び出して、作業中のアプリへ貼り付けるツールです。

- Dock には表示されません
- メニューバー右上の **クリップ（📎）アイコン** から操作します
- コピーしたテキストは自動で履歴に記録されます（Clipy と同等の使い勝手）

---

## 2. 初回セットアップ

### 2.1 インストール

1. [README.md](../README.md) の手順で `/Applications/Snippet Manager.app` に配置
2. アプリを起動（`open -a "Snippet Manager"`）

### 2.2 必須権限: アクセシビリティ

自動ペーストには **アクセシビリティ** の許可が必要です。

1. システム設定 → **プライバシーとセキュリティ** → **アクセシビリティ**
2. **Snippet Manager** をオン

許可しない場合、スニペットはクリップボードにコピーされますが、**自動で ⌘V が送られません**。

### 2.3 ログイン時に起動（任意）

1. メニューバー → **環境設定…**（⌘,）
2. **一般** タブ → **ログイン時に起動** をオン
3. 初回は macOS の承認が必要な場合があります → **ログイン項目を開く** から許可

## 3. 履歴・スニペットの呼び出し

### 3.1 ホットキー

| 項目 | 初期値 | 内容 |
|------|--------|------|
| メインメニューを開く | `⌘⇧V` | クリップボード履歴 + スニペット |
| 履歴メニューを開く | `⌘⌃V` | クリップボード履歴のみ |
| スニペットメニューを開く | `⌘⇧B` | スニペットのみ |

変更する場合: **環境設定 → ショートカット** で Recorder をクリックし、新しいキーを入力。

### 3.2 メニュー操作

1. 貼り付け先アプリでカーソルを置く
2. ホットキーを押す → **マウス位置** にメニューが表示される
3. 履歴は新しい順に番号付きで並ぶ。フォルダ名をクリックするとサブメニューが開く
4. 項目をクリック、または数値キーで選択 → 自動でペースト

| キー | 動作 |
|------|------|
| `1`〜`9` | 1〜9 番目の項目 |
| `0` | 10 番目の項目 |
| `Esc` | メニューを閉じる |

メニューバーアイコンからも同じ内容（履歴 + スニペット階層）にアクセスできます。

### 3.3 クリップボード履歴について

- どのアプリでも **⌘C したテキストが自動で記録** されます（デフォルト 30 件まで）
- 直近 10 件はメニュー直下に、それ以前は「11 - 20」のようなフォルダにまとまります
- 同じテキストを再コピーすると先頭に移動します（重複して増えません）
- パスワードマネージャ等が秘匿指定したコピーは記録されません
- 履歴はアプリを再起動しても保持されます
- 件数などの調整: **環境設定 → 履歴**（§5.1）

## 4. スニペットの管理

**環境設定 → スニペット** タブ、またはメニュー **スニペットを編集**（⌘E）

### 4.1 画面構成

```
┌─────────────────────────────────────────────┐
│ [フォルダを追加] [スニペットを追加] [削除]    │  ← ツールバー
├──────────┬──────────────────────────────────┤
│ ツリー    │ 編集ペイン                        │
│ フォルダ  │ ・フォルダ選択時 → フォルダ名       │
│  └スニペ │ ・スニペット選択時 → タイトル・本文 │
└──────────┴──────────────────────────────────┘
```

### 4.2 フォルダ

| 操作 | 手順 |
|------|------|
| 作成 | **フォルダを追加** → 右ペインで名前を編集 |
| 名前変更 | 左でフォルダを選択 → 右の「フォルダ名」を編集 |
| 削除 | フォルダを選択 → **削除** または Delete キー（中のスニペットも削除） |

### 4.3 スニペット

| 操作 | 手順 |
|------|------|
| 追加 | フォルダを選択 → **スニペットを追加** または **⌘N** |
| 編集 | スニペットを選択 → 右でタイトル・本文を編集（自動保存） |
| 別フォルダへ移動 | 左のスニペットを **ドラッグ＆ドロップ** |
| 並べ替え | 同一フォルダ内で行の間にドロップ |
| 削除 | スニペットを選択 → **削除** または Delete キー |

### 4.4 保存について

すべての変更は **即座に UserDefaults へ保存** されます。明示的な「保存」ボタンはありません。

## 5. メニューバー一覧

| メニュー項目 | ショートカット | 説明 |
|-------------|---------------|------|
| 履歴（番号付き） | — | クリップボード履歴（新しい順） |
| スニペット（階層） | — | フォルダ別スニペット一覧 |
| 履歴を消去 | — | 確認のうえ履歴を全削除 |
| スニペットを編集 | ⌘E | 環境設定のスニペットタブ |
| 環境設定… | ⌘, | 一般・ショートカット・履歴・スニペット |
| Snippet Manager を終了 | ⌘Q | アプリ終了 |

### 5.1 環境設定 → 履歴タブ

| 設定 | 初期値 | 範囲 |
|------|--------|------|
| 履歴の最大保存件数 | 30 件 | 1〜100 |
| メニューに直接表示する件数 | 10 件 | 0〜20 |
| フォルダあたりの件数 | 10 件 | 1〜20 |

「履歴を消去…」ボタンで全履歴を削除できます（確認ダイアログ付き）。

## 6. トラブルシューティング

### メニューバーにアイコンが出ない

- アプリが起動しているか確認: `pgrep -x "Snippet Manager"`
- メニューバーが溢れている → `Control` + メニューバー左端ドラッグで隠れていないか確認
- 再インストール後は一度終了してから起動: `pkill -x "Snippet Manager"; open -a "Snippet Manager"`

### ホットキーが効かない

- **環境設定 → ショートカット** で設定を確認
- 他アプリと競合していないか確認
- **入力監視** 権限が必要な場合あり（システム設定 → プライバシーとセキュリティ → 入力監視）

### ペーストされない

- **アクセシビリティ** がオンか確認（§2.2）
- 貼り付け先アプリがペーストを許可しているか確認

### フォルダを作っても何もできないように見える

- フォルダ作成後、**左のツリーでフォルダが選択** されているか確認
- 右ペインでフォルダ名を編集できる
- **スニペットを追加**（⌘N）または他フォルダからドラッグでスニペットを入れる

---

## 7. 機能範囲（参考）

次の機能は含みません。

- クリップボード履歴の画像・書式付きテキスト・ファイル対応（テキストのみ）
- スニペットのインポート / エクスポート
- フォルダごとの個別グローバルホットキー
